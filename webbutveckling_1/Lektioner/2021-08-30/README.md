## Repetition

Vad krävs minst i ett HTML dokument?

```html
<!DOCTYPE html>
<html>
<head>

</head>
<body>

</body>
</html>
```

Minsta sida som visar något:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Min första hemsida</title>
</head>
<body>
<h1>Med en header</h1>
<p>Och en brödtext</p>
</body>
</html>
```
