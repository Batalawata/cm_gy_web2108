## Ny HTML

### Tabeller

Taggar:

- `<table>` - Skapa tabell.
- `<tr>` - Table Row (Tabell Rad)
- `<th>` - Table Header (Tabell titel)
- `<td>` - Table Data (Tabell Cell)

```html

<table>
    <tr>
        <th>Company</th>
        <th>Contact</th>
        <th>Country</th>
    </tr>
    <tr>
        <td>Alfreds Futterkiste</td>
        <td>Maria Anders</td>
        <td>Germany</td>
    </tr>
    <tr>
        <td>Centro comercial Moctezuma</td>
        <td>Francisco Chang</td>
        <td>Mexico</td>
    </tr>
</table>
```
